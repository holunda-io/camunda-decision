# camunda-decision


Extension for easier, type safe interaction with camunda dmn





## DmnWriter

Prints ascii table like:

├───────────────────┴───────────────────┼───────────────────┼──────────────────┤
│                 INPUT                 │      OUTPUT       │   DESCRIPTION    │
├┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┼┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┼┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┤
